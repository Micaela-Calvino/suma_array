# Challenge Técnico: Suma de dos elementos en un array

Este proyecto contiene la solución a un ejercicio técnico propuesto por Pigmalion Software para el puesto de desarrolladora full-stack.

---

## Problema

Dado un array de números enteros y un número objetivo (`requiredSum`), determinar si existen dos elementos en el array cuya suma sea igual a `requiredSum`.

---

## Archivos

- `suma.py`: Contiene dos implementaciones de la solución:
  - `suma_simple`: Algoritmo sencillo que recorre todas las combinaciones posibles.
  - `suma_eficiente`: Algoritmo optimizado que utiliza un conjunto para mejorar el tiempo de ejecución.

- `test_suma.py`: Pruebas automáticas con `assert` para verificar el correcto funcionamiento de ambas funciones.

---

## Uso

1. Ejecutar el archivo `test_suma.py` para validar que ambas funciones funcionan correctamente.
2. Modificar o ampliar el código según sea necesario.

---

## Tecnologías y versiones

- Lenguaje: Python 3.12.6
- Entorno: cualquier editor de texto o IDE que soporte Python
